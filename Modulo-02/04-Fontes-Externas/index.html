Título

Para usar uma fonte externa baixada ou especifica, geralmente vai estar "otf" ou "ttf". Digamos que um cliente solicitou uma fonte diferente, que não temos no google fonts, uma outra opção seria o site "dafont.com", lá tem varias opções de tematicas. Nele, precisaremos baixar a font, descompactar da pasta zip, muitas vezes vira no formato otf e ttf. Devemos fazer uma pasta para colocar elas no visual studio code. E na opção style ou no css separado devemos usar o "@font-face", que serve para incorporar fontes personalizadas de um site.

Apos colocar o @font-face, dentro do style ou no css separado, aparecerá:

  • font-family: (onde devemos colocar um nome para determinada fonte)
  • src: url ( ) - (onde devemos colocar a fonte que foi baixada, nome exatamente igual do arquivo)

Devemos acrecentar:

  • Format ( )
  • sendo: opentype (otf),truetype (ttf), embedded-opentype, truetype-aat (Apple Advanced Typography), sendo o otf e ttf mais utiliados

E caso tenha varios tipos de formatos (ex: otf, ttf, entre outros), podemos acrecentar outras URL( ) e outros Format( ), pois assim se adequara a varios tipos de navegadores.

Antes de baixar as fontes, lembrar de verificar nas configurações das mesmas, se possuem acentuação!
